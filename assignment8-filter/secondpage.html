Dashboard gives information about each parking area like total number of slots in a parking area, number of slots reserved, occupied slots, available/free slots

Reserved Slots
This gives information about the total number of slots reserved for employees in a parking area
Occupied Slots
This gives information about the total number of vehicles present in the parking area
Available Slots
This gives information about the total number of free slots available for parking excluding the reserved slots
